# Scripts Directory

Organized collection of scripts for Onasis-CORE operations.

## Directory Structure

### `test/`
Test scripts for verifying functionality:
- MCP server connection tests
- API endpoint tests
- Memory operations tests
- End-to-end integration tests

### `setup/`
Setup and initialization scripts:
- Memory submodule configuration
- GitHub remote setup
- Complete environment setup

### `migration/`
Database migration scripts:
- Neon DB migrations
- Supabase migrations
- Schema updates

### `deployment/`
Deployment scripts (to be added)

### `fix/`
Emergency fix scripts:
- Credential scrubbing
- Security patches

## Root Scripts

Utility scripts in this directory:
- `verify-vps-services.sh` - VPS health check
- `verify-config.sh` - Configuration validation
- `REORGANIZE_ONASIS_CORE.sh` - Codebase reorganization
- `create-oauth-issues.sh` - OAuth issue tracker
- `enhance-extensions-mcp.sh` - MCP enhancement script
- `external-mcp-client.js` - External MCP client
- `multi-platform-router.js` - Multi-platform routing
- `unified-router.js` - Unified routing logic
- `ai-service-router.js` - AI service routing
- `cli-integration.js` - CLI integration utilities
- `vps-backup-solution.sh` - VPS backup automation
